TECHNICAL SPECIFICATION
Project: Expiry Tracker
Version: MVP (v1)

========================================
1. PROJECT OVERVIEW
========================================

Expiry Tracker is a lightweight, open-source Management Information System (MIS) web application for tracking items with expiration or renewal dates.

The application solves one problem:

"Help users know what has expired, what is expiring soon, and what remains active."

Version 1 intentionally focuses on a minimal, maintainable MVP.

Excluded from v1:
- Authentication
- Notifications
- File uploads
- Integrations
- Reports and analytics
- Team collaboration
- Roles and permissions
- Background jobs
- APIs for third parties

========================================
4. DATABASE DESIGN
========================================

Table: items

Fields

- id (UUID)
- title (String)
- category (String)
- expiration_date (Date)
- notes (Text, optional)
- created_at (Timestamp)
- updated_at (Timestamp)

Status is NOT stored.

Status is calculated dynamically from:

Current Date
+
Expiration Date

========================================
5. STATUS RULES
========================================

Expired
Days Remaining < 0

Expiring Soon
Days Remaining >= 0
AND
Days Remaining <= 30

Active
Days Remaining > 30

========================================
9. EDGE CASES
========================================

- Invalid dates
- Empty title
- Duplicate entries
- Expiration today
- Leap year dates
- Empty database
- Empty search results
- Very old expiration dates
- Long titles
- Delete confirmation

Store dates only (not timestamps) to avoid timezone issues.

========================================
12. ENGINEERING PRINCIPLES
========================================

- Keep the architecture simple.
- Prefer calculated values over stored values.
- Solve one problem well.
- Avoid feature creep.
- Build for maintainability.
- Make the project easy to understand, contribute to, and self-host.
